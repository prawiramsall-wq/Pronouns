📘 Personal Pronouns

Complete the exercises below to practice using personal pronouns. Personal pronouns are used in place of nouns. For example, consider the following sentence: “The woman is pretty.” We can write the same sentence using a pronoun: “She is pretty.”

Personal pronouns can be difficult. Each pronoun must match the verb conjugation. For example, when using the verb “to be,” the conjugation changes. When using the pronoun “I,” the verb form becomes “am”: “I am young.” When using the pronoun “she,” the verb form becomes “is”: “She is young.”

Examples: I am, You are, He is, She is, It is, We are, They are
